# Laravel Task Manager

Este proyecto es un **Task Manager** hecho con **Laravel** y **MySQL**.

## Funcionalidades

- Crear tareas
- Listar tareas
- Marcar tareas como completadas
- Eliminar tareas

## Rutas de prueba

- `/create-task` → crea una tarea de prueba
- `/tasks` → lista todas las tareas
- `/tasks/{id}/complete` → marca una tarea como completada
- `/tasks/{id}/delete` → elimina una tarea

## Tecnologías

- PHP / Laravel
- MySQL
- Git / GitHub
